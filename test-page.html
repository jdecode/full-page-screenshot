Test Page for Screenshot Extension

Section 1: Welcome

This is a test page for the Full Page Screenshot Chrome extension. The page is designed to be taller than a typical viewport to test the full page screenshot functionality.

Section 2: Features

When you take a screenshot of this page, the filename should be generated as:

test-page-for-full-page-screenshot-YYYYMMDD-HHMMSS.png

Where spaces are replaced with dashes and the timestamp includes seconds.

Section 3: Instructions

  1. Load this extension in Chrome (chrome://extensions/)
  2. Enable Developer Mode
  3. Click "Load unpacked" and select the extension directory
  4. Navigate to this page
  5. Click the extension icon and take a screenshot

Section 4: Bottom Content

If you can see this section in your screenshot, the full page capture is working correctly!

The screenshot should include all content from top to bottom.
